# Оно тебе надо

Лэндинг аукциона вещей, в которые никто не верил.

## Описание проекта

«Оно тебе надо» — это аукционная площадка для необычных лотов: дневники подростков, картины начинающих художников, книги непризнанных авторов. Вещи, которые казались неудачными, но стали культовыми.

## Технологии

- HTML5
- CSS3 (Flexbox, Grid)
- БЭМ методология

## Структура проекта

```
ono-tebe-nado-fd/
├── fonts/              # Шрифты проекта
├── images/             # Изображения и иконки
├── styles/
│   ├── global.css      # Глобальные стили
│   └── style.css       # Основные стили
└── index.html          # Главная страница
```

## Функциональность

- ✅ Адаптивная верстка
- ✅ Семантическая разметка
- ✅ Hover-эффекты на интерактивных элементах
- ✅ Карточки лотов с overlay
- ✅ Формы для связи (телефон, email)

## Секции

1. **Header** — навигация, логотип, контакты
2. **Cover** — главный экран с заголовком и призывом к действию
3. **Lots** — карточки с лотами аукциона
4. **About** — информация об аукционе
5. **Footer** — контакты, навигация, социальные сети

## Установка и запуск

Проект не требует установки зависимостей. Просто откройте `index.html` в браузере.

```bash
# Клонировать репозиторий
git clone <repository-url>

# Перейти в папку проекта
cd ono-tebe-nado-fd

# Открыть в браузере
open index.html
```
